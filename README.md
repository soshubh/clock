# Dot Clock

A fullscreen dot-matrix LED clock with customizable pitch, size, glow, accent color, and 12H/24H format.

**Live:** https://clock.whimsaas.online/

Open `index.html` in a browser, or deploy the static files to any host.

## SEO files

- `sitemap.xml` — submit to Google Search Console
- `robots.txt` — crawler rules and sitemap reference
- `site.webmanifest` — PWA-style app metadata
